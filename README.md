# -Smart-Healthcare-Monitoring
An Optimized CNN-GRU Deep Learning Framework for Smart Healthcare Monitoring Using Wearable Sensor Data


Smart health monitoring systems have become essential for real-time diagnosis and timely medical intervention due to advances in wearable sensors and IoT-based healthcare technologies. Traditional machine learning and deep learning approaches, such as CNN and LSTM, often struggle to capture complex spatiotemporal dependencies in physiological signals, leading to delayed anomaly detection and higher false alarm rates. This study proposes an intelligent CNN-GRU framework that combines convolutional neural networks for spatial feature extraction with gated recurrent units for temporal dependency learning from multimodal health sensor data. To further improve performance, the Sea Lion Optimization (SLO) algorithm is applied for hyperparameter tuning, while Z-score normalization ensures data consistency and faster convergence. The proposed model was evaluated using a publicly available wearable healthcare dataset and achieved an accuracy of 99.70%, outperforming traditional CNN and CNN-LSTM models. The framework demonstrated superior precision, recall, and F1-score values, confirming its effectiveness for accurate, reliable, and real-time health anomaly detection in smart healthcare systems.


Objectives
Come up with a deep hybrid learning architecture that uses the guts of CNN and GRU models in extracting spatial-temporal features from smart sensor healthcare data.
Normalize the data by applying Z-Score normalization in data preprocessing of multi-modal inputs and subsequently improve the model performance and training stability of the deep-learning. 
Increase the accuracy for classifying known data into the specified classes by optimizing the hyperparameters of the CNN-GRU model using the SLO Algorithm.
Create a system that can watch a person's health and then probably notify him or her of an anomaly when it occurs in the process of monitoring for early diagnosis and intervention.
